- **Highlights:** Compared declarative vs imperative approaches, with notes on extending multi-cloud orchestration.

### 3. Jenkins + Docker Install Script (Ubuntu 24.04)

- **Tech Stack:** Bash, Docker, Jenkins, Ubuntu
- **Highlights:** Troubleshooting logs included; ideal for onboarding new engineers or testing pipeline compatibility.
